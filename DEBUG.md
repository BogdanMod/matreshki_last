# Как открыть консоль браузера на телефоне

## Для Android (Chrome)

### Способ 1: Chrome DevTools (USB отладка)

1. **На телефоне:**
   - Откройте **Настройки** → **О телефоне**
   - Нажмите 7 раз на **Номер сборки** (активирует режим разработчика)
   - Вернитесь в **Настройки** → **Для разработчиков**
   - Включите **Отладка по USB**

2. **На компьютере:**
   - Подключите телефон через USB
   - Откройте Chrome на компьютере
   - Перейдите по адресу: `chrome://inspect`
   - Найдите ваш телефон в списке
   - Нажмите **Inspect** рядом с открытой вкладкой сайта

3. **В DevTools:**
   - Откройте вкладку **Console** для просмотра логов
   - Вкладка **Network** покажет загрузку ресурсов
   - Вкладка **Elements** покажет HTML структуру

### Способ 2: Chrome Remote Debugging (Wi-Fi)

1. **На телефоне:**
   - Откройте Chrome
   - Введите в адресной строке: `chrome://inspect`
   - Включите **Discover USB devices**

2. **На компьютере:**
   - Откройте Chrome → `chrome://inspect`
   - Убедитесь, что телефон и компьютер в одной Wi-Fi сети
   - Найдите телефон и нажмите **Inspect**

## Для iPhone/iPad (Safari)

### Способ 1: Safari Web Inspector (USB)

1. **На iPhone:**
   - Откройте **Настройки** → **Safari** → **Дополнения**
   - Включите **Веб-инспектор**

2. **На Mac:**
   - Подключите iPhone через USB
   - Откройте Safari на Mac
   - В меню: **Разработка** → выберите ваш iPhone → выберите открытую вкладку
   - Откроется Web Inspector с консолью

### Способ 2: Safari Web Inspector (Wi-Fi)

1. **На iPhone:**
   - Откройте **Настройки** → **Safari** → **Дополнения**
   - Включите **Веб-инспектор**

2. **На Mac:**
   - Убедитесь, что iPhone и Mac в одной Wi-Fi сети
   - В Safari на Mac: **Разработка** → выберите iPhone → выберите вкладку

## Альтернативные способы (без компьютера)

### 1. Использование Eruda (консоль прямо на странице)

Добавьте в начало `index.html`:

```html
<script src="https://cdn.jsdelivr.net/npm/eruda"></script>
<script>eruda.init();</script>
```

Это добавит иконку консоли в правом нижнем углу страницы.

### 2. Использование vConsole

Добавьте в начало `index.html`:

```html
<script src="https://unpkg.com/vconsole@latest/dist/vconsole.min.js"></script>
<script>new VConsole();</script>
```

### 3. Просмотр логов через alert()

Временно добавьте в код:

```javascript
console.log = function(...args) {
  alert(JSON.stringify(args));
};
```

## Быстрый способ для отладки прямо сейчас

Я добавил компонент отображения ошибок прямо на странице. Если произойдет ошибка, она будет показана на экране с кнопкой "Перезагрузить".

Также все ошибки логируются в консоль браузера, которую можно открыть одним из способов выше.
